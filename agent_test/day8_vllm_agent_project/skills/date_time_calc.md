# date_time_calc

## 作用
基于 mock 城市时间做确定性的时间偏移计算。

## 何时使用
- 用户问“几小时后 / 几天后是什么时间”
- 用户先查当前时间，再要求做时间偏移

## 何时不要使用
- 用户只是问当前时间，不需要偏移
- 用户给出的城市不在支持列表内

## 参数格式
```json
{
  "city": "东京",
  "delta": 3,
  "unit": "hour"
}
```

## 成功返回示例
```json
{
  "ok": true,
  "data": {
    "city": "东京",
    "base_time": "2026-04-15 11:30:00",
    "delta": 3,
    "unit": "hour",
    "result_time": "2026-04-15 14:30:00"
  },
  "error": null
}
```

## 常见组合
- `get_time -> date_time_calc`

## 标准例句
1. 先告诉我东京现在几点，再算东京 3 小时后是几点。
2. 北京 2 天后是几号？
3. 纽约 5 小时前大概是什么时间？
